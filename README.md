# concept-peggle-game

## Fabriek-setting waar je probeert pegs te schieten die punten opleveren die je kan gebruiken voor upgrades

# upgrade concepten

## bom (grote explosie die meerdere pegs raakt)
## sticky (de bal plakt aan de peg om de peg 2 keer te raken)
## clone (schiet 2 ballen)
## bounce (als de bal een peg raakt schiet die een kogel naar een andere peg)
## multiply (pegs kunnen zichzelf klonen op andere plekken op het bord)
## damage (de bal raakt de peg 2 keer per aanraking)


# hoe speelt het

## je schiet de bal naar de pegs en de upgrades die je hebt worden geactiveerd, tussen schoten door kan je naar de schop om meer upgrades te kopen
